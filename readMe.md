# Authentication with outlook API

# Requirements

- nodejs
- npm
- express
- express-session
- passport
- passport-outlook

# Usage

```cmd
# install dependances

npm i ou npm install

# after install dependencies run app

npm run start
```

go to `http://localhost:3000/` to use application<br><br>
